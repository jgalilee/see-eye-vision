Class: module:app/Renderer

module:app/Renderer(id, renderer, onReady)

Wrapper around the Renderer IPC. It primarily acts as a buffer to messages sent from the IPC before the Main IPC is ready.

Constructor

new module:app/Renderer(id, renderer, onReady)

Creates the wrapper and when the document is ready sends any messages that were sent / received before the document was ready.
Parameters:
Name Type Description
id String Identifier for the renderer IPC messages.
renderer * Renderer IPC object.
onReady function Callback to mark the renderer as ready, defaults to onDocumentReady.
